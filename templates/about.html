About This Application

This Breast Cancer Detection System uses machine learning to predict whether a tumor is benign or malignant based on input features derived from a biopsy.

The goal is to assist medical professionals with faster, more consistent diagnostic support. However, this tool is not a replacement for professional medical advice or testing.
